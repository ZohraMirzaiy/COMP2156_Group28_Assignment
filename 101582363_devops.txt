COMP2156 DevOps for System Administration introduces the core ideas of modern DevOps practice. In this course we learn how to use tools like Git, GitHub, GitHub Actions, continuous integration pipelines, and automation scripts. The labs are focused on real workflows such as branching, pull requests, code reviews, and automated builds. By combining version control with CI, we explore how to keep software reliable while teams work in parallel. This course is important because it connects system administration skills with software delivery, which is exactly what many junior DevOps roles expect.
We also learn how to use GitHub Actions to automate builds and tests.
Assignments and labs are designed to simulate real DevOps tasks.
